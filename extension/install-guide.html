FFmpeg Setup Guide

This app requires FFmpeg to compress and convert videos

FFmpeg Is Required

FFmpeg Compressor cannot compress or convert video without FFmpeg. The extension uses FFmpeg on your own device so files stay local and your computer does the processing. If the app says FFmpeg is missing or could not start, follow this guide before trying another job.

Install FFmpeg On Windows

  1. Open the official FFmpeg download page.
  2. Under Windows EXE Files, choose the gyan.dev Windows build.
  3. Download the release build zip.
  4. Extract the zip to a simple folder such as C:\ffmpeg.
  5. Add the extracted bin folder to your Windows PATH. It will look similar to C:\ffmpeg\bin.
  6. Close and reopen PowerShell or Windows Terminal.
  7. Check that FFmpeg is installed and ready:
ffmpeg -version
Open Official FFmpeg Downloads

Optional Quick Install

If Windows already has winget available, this is faster. If the command is not recognized, use the official download steps above. 

winget install --id Gyan.FFmpeg --exact

If FFmpeg Still Is Not Ready

  • Restart Chrome and reopen the extension.
  • Restart your terminal so it gets the updated PATH.
  • Run ffmpeg -version again to confirm Windows can find it.
  • Make sure the PATH entry points to the folder that contains ffmpeg.exe, not just the extracted parent folder.
